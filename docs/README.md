# Ruby WASM Todo Showcase

Welcome to the **Ruby WASM Todo** documentation! This project is a production-grade demonstration of running Ruby 4.0 (WebAssembly) directly in the browser with full Gem support.

## 🚀 Vision
To showcase that Ruby is not just for the server. With WebAssembly, Ruby becomes a powerful tool for client-side logic, enabling:
- **Shared Logic**: Run the exact same Ruby code on frontend and backend.
- **Rich Ecosystem**: Use existing Ruby gems (like Faker, Rouge, etc.) in the browser.
- **High Performance**: Leverage binary execution through WASM.

## 🛠️ Tech Stack
- **Ruby 4.0 WASM**: The core engine running in the browser.
- **Vanilla JS**: Modulo-based architecture for the bridge.
- **Vanilla CSS**: Premium dark-mode design with Glassmorphism.
- **DocsViewer**: Our bespoke markdown presentation system.
